# Atri Fashion MNIST Assignment

This project contains a simple neural network implementation from scratch using NumPy to classify the Fashion-MNIST dataset.

## Structure

- `src/neural_network.py`: Contains the `NeuralNetwork` class with forward and backward propagation.
- `notebooks/`: Contains Jupyter notebooks.
  - `Task1.ipynb`: For data exploration and visualization.
  - `test_nn.ipynb`: For training and testing the neural network.
- `requirements.txt`: Contains the project dependencies.
- `plots/`: Contains saved plots.

## How to run

1.  Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```

2.  Run the notebooks in the `notebooks/` directory. Start with `Task1.ipynb` to see the data, and then run `test_nn.ipynb` to train and evaluate the model.
